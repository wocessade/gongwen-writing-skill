# 公文写作 Skill — GB/T 9704-2012

党政机关公文写作的 [Claude Code](https://claude.ai/code) 技能模块，严格遵循 **GB/T 9704-2012《党政机关公文格式》** 标准。

## 能力

- **撰写新公文**: 支持 15 种法定文种（通知、报告、请示、批复、函、纪要等）
- **格式审查**: 自动检查 .docx 是否符合 GB/T 9704-2012，含语义级字体/字号/行距校验
- **格式转换**: 将非标文档转换为标准公文格式

## 快速开始

在 Claude Code 中直接输入：

```
请帮我写一份关于做好2024年元旦春节期间有关工作的通知，发文机关是XX省人民政府办公厅
```

或检查已有公文：

```
请检查这份公文是否符合GB/T 9704-2012格式标准: /path/to/document.docx
```

## 前置条件

- Python 3.8+ with `python-docx`
- 推荐安装字体：方正小标宋简体、仿宋_GB2312、楷体_GB2312、黑体
  （缺省时自动回退并给出警告）

## 管道流程

```
G0(文种确认) → G1(结构规划) → G2(起草.docx) → G3(格式审查) → G4(修订收敛)
```

支持全自动和半自动两种模式。

## 项目结构

| 目录 | 用途 |
|------|------|
| `strategists/` | G0-G4 各阶段策略指令 |
| `scripts/` | docx 生成、格式转换、发文处理笺脚本 |
| `evaluate/` | 格式审查子系统（LLM agent 提示词 + 自动校验脚本） |
| `references/` | GB/T 9704 规范、文种模板、版记指南 |
| `templates/` | 常用公文模板（JSON） |
| `composers/` | docx 组装规范 |
| `static/core/` | 管道核心规则 |
| `auto_mode/` | 全自动模式编排器 |
