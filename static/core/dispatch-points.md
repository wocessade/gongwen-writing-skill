| DP25 | S1 研究问题冗余 | **Mode B (×3 redundancy):** 3 agents independently generate a research question from the same topic + background. All receive identical input: topic description, FINER criteria (Feasible, Interesting, Novel, Ethical, Relevant), output format (one-sentence question + 3-sentence rationale). After collection: compare for overlap — if 2/3 converge, accept as `[AGENT-CONSENSUS × 2/3]`. If all 3 diverge → LOW agreement → present all 3 to user. **Note:** DP25 (Mode B ×3 redundancy) is for Idea-First entry only. Data-First entry uses single-agent decision tree in D2C — data constraints provide natural validation. | 3 | ★★★ |
| DP26 | S2/S2-LR 研究空白识别 | **Mode B (×2 redundancy):** 2 agents independently identify the research gap from the same literature set. Both receive: research question, bibliography summary, citation network clusters. Output format: gap statement (1 paragraph) + supporting evidence (2-3 papers). After collection: compare gap statements. HIGH agreement → accept. PARTIAL → orchestrator merges. LOW → user decides. | 2 | ★★★ |
| DP27 | S2.5 创新性评分 | **Mode B (×2 redundancy):** 2 agents independently score novelty across 6 dimensions (topic, method, data, theory, application, scope) on 1-5 scale. Both receive: research question, gap analysis, competing papers table. Output: 6 scores + 1 sentence justification each. After collection: compare per-dimension. Divergence > 1 on any dimension → flag for orchestrator review. Average scores for final. | 2 | ★★★ |
| DP28 | S8.5 期刊推荐 | **Mode B (×3 redundancy):** 3 agents independently recommend 3-5 target journals. All receive: paper abstract, type (empirical/review/theory), reference count, writing format, key constraints (open access, APC budget, timeline). Output: ranked list + rationale per journal + scope match score. After collection: intersection = high-confidence recommendations. Journals appearing in 2/3 = secondary. 1/3 = list but flag. | 3 | ★★★ |
| DP29 | T2 GAP 矩阵填充 | **Mode B (×2 redundancy):** 2 agents independently fill the research gap matrix (research-questions × methodological-approaches). Both receive: T2E merged bibliography, thesis topic, research sub-questions. Output: filled matrix with counts in each cell. After collection: compare cell-by-cell. Cells where both agents agree → accept. Cells differing by >2 or one agent found a paper the other missed → flag for manual review. | 2 | ★★★ |
| DP30 | S9 审稿人冗余验证 | **Mode C (hybrid):** 4 reviewer personas (methodologist, domain expert, skeptic, generalist) each run with ×2 redundancy. 8 agents total (4 personas × 2). Each pair receives the same manuscript + persona description. After collection: per-persona compare for omissions (items one reviewer caught but the other missed). Merged report keeps union of all findings. Findings flagged if only 1 of 2 reviewers in the same persona noted it — those are higher miss-risk. | 8 | ★★★ |
