# 公文语言规范指南

## 基本原则

公文语言应做到：准确、简明、庄重、规范。

| 原则 | 说明 |
|------|------|
| 准确 | 概念明确、判断正确、措辞恰当、不产生歧义 |
| 简明 | 言简意赅、删繁就简、力戒空话套话 |
| 庄重 | 风格严肃、格调庄重、客观冷静 |
| 规范 | 使用标准书面语、符合语法规范、使用规范简称 |

---

## 常用公文开头用语

| 场景 | 用语 |
|------|------|
| 为…（目的） | "为"、"为了"、"为进一步" |
| 根据/依据 | "根据"、"依据"、"按照"、"遵照" |
| 鉴于 | "鉴于"、"考虑到" |
| 经…（批准） | "经"、"已经"、"业经"、"经研究" |
| 现将/现就 | "现将"、"现就"、"现予" |

## 常用公文结尾用语

| 文种 | 结尾用语 |
|------|---------|
| 通知 | "特此通知"、"请遵照执行" |
| 报告 | "特此报告"、"以上报告，请审阅" |
| 请示 | "妥否，请批示"、"请予审批" |
| 批复 | "此复"、"特此批复" |
| 函(去函) | "盼复"、"请予函复" |
| 函(复函) | "特此函复" |
| 通报 | "特此通报" |
| 通告 | "特此通告" |

---

## 常见语言问题及纠正

| 问题类型 | 不当表达 | 规范表达 |
|---------|---------|---------|
| 口语化 | "这事儿我们研究过了" | "此事已研究" |
| 啰嗦 | "为了更好地促进" (删"更好地") | "为促进" |
| 歧义 | "超过三个部门提出意见" | "三个以上部门提出意见" |
| 夸张 | "造成了极其严重的影响" | "造成不良影响" |
| 情绪化 | "令人十分遗憾" | "应予纠正" |
| 绝对化 | "一律全部取消" (一律/全部重复) | "一律取消" |
| 人称不当 | "你们要高度重视" | "请各单位高度重视" |
| 语气不当 | "必须立即停止" (语气过强) | "应立即停止" |

## 数字用法规范

| 规则 | 示例 |
|------|------|
| 公历年月日用阿拉伯数字 | 2024年1月15日 |
| 百分比用阿拉伯数字 | 同比增长15% |
| 成文日期用阿拉伯数字 | 2024年1月15日 |
| 发文字号中年份用阿拉伯数字+六角括号 | X政发〔2024〕1号 |
| 惯用语用汉字 | "五四运动"、"十四五规划" |
| 星期几用汉字 | "星期一" |
| 专用名称中的数字用汉字 | "第九号令" |
| 概数用汉字 | "三五天"、"十三四岁" |

## 标点符号规范

- 主送机关名称后用全角冒号
- 正文标题层级中：一级 "一、"（顿号），二级 "（一）"（无标点），三级 "1."（下角点）
- 附件名称后不加标点
- 引用文件标题用书名号《》

---

## 政治性语言规范

1. **领导同志称谓规范**：全称或规范化简称，不加"同志"外的修饰语
2. **机构名称规范**：使用全称或规范化简称，首次出现使用全称，后文可用括号注明规范简称
3. **政治术语准确**：严格使用中央和上级文件中的规范表述
4. **时事政策引用**：必须准确、完整，注明出处和发文日期

## 常见公文句式

| 句式 | 用途 | 示例 |
|------|------|------|
| "要…" | 要求性 | "要高度重视此项工作" |
| "应当…" | 规范性 | "应当严格执行相关规定" |
| "严禁…" | 禁止性 | "严禁弄虚作假" |
| "鼓励…" | 倡导性 | "鼓励探索创新方式方法" |
| "按照…要求" | 依据性 | "按照省委省政府部署要求" |
