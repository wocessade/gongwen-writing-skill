# 公文写作 Skill — 术语表

## 通用管道术语

参见 `../academic-shared/GLOSSARY.md` 了解以下术语：
- Gate (门控)、Gate Checklist、Gate Failure Route
- STOP-AND-ASK
- Convergence Loop (收敛循环)
- Degradation Detection (退化检测)

## 公文特有术语

| 术语 | 说明 |
|------|------|
| **文种** | 15种法定公文类型（通知、报告、请示、批复、函、纪要、通报、通告、公告、决定、决议、命令、公报、意见、议案） |
| **红头** | 发文机关标志，红色方正小标宋字体，位于版心上部，下方有红色分隔线 |
| **红色反线** | 红头下方的红色分隔线（width: 全文宽, thickness: 0.35mm-0.5mm） |
| **发文字号** | 由发文机关代字、年份、发文顺序号组成，如 "X政发〔2024〕1号" |
| **签发人** | 上行文（报告、请示）必须标注的签发人姓名 |
| **主送机关** | 公文的主要受理机关，应当使用全称或者规范化简称 |
| **抄送机关** | 除主送机关外需要执行或者知晓公文内容的机关 |
| **成文日期** | 公文的生效日期，用阿拉伯数字将年、月、日标全 |
| **骑车盖月** | 印章的上边缘压住日期（不压正文），下边缘盖在日期上 |
| **上行文** | 下级机关向上级机关报送的公文（报告、请示） |
| **下行文** | 上级机关向下级机关发布的公文（通知、批复、决定等） |
| **平行文** | 同级或不相隶属机关之间的公文（函） |
| **版头** | 公文首页红色分隔线以上的部分 |
| **主体** | 红色分隔线以下、版记以上的部分 |
| **版记** | 公文末页首条分隔线以下、末条分隔线以上的部分 |
